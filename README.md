## Frameworks and IDE

- React Hooks
- Redux
- Material UI
- Visual Studio Code

## Build and Run

1. Git clone the project
2. Checkout master and pull latest
3. Open VS Code Terminal
4. Run command `yarn`
5. Run command `yarn start`
6. App opens in browser with http://localhost:3000
